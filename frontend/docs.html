API Documentation

Integrate sports numerology predictions into your applications

Base URL

https://crusty-betting.up.railway.app

Authentication

All API requests (except demo) require an API key in the header:

X-API-Key: your_api_key_here

Endpoints

POST /api/v1/demo-analyze

Free demo analysis (5 per IP per day)

Request Body

{
  "player1_name": "Novak Djokovic",
  "player1_birthdate": "1987-05-22",
  "player2_name": "Carlos Alcaraz",
  "player2_birthdate": "2003-05-05",
  "match_date": "2026-02-28",
  "sport": "tennis"
}

Response

{
  "match_date": "2026-02-28",
  "sport": "tennis",
  "player1": {
    "name": "Novak Djokovic",
    "life_path": 7,
    "score": 21
  },
  "player2": {
    "name": "Carlos Alcaraz",
    "life_path": 6,
    "score": 0
  },
  "winner_prediction": "Novak Djokovic",
  "confidence": "STRONG",
  "remaining_tries": 4
}

POST /api/v1/analyze-match

Full analysis with API key (10 per day)

Headers

X-API-Key: sn_your_api_key

Request Body

Same as demo endpoint

GET /api/v1/demo-status

Check remaining demo tries

Response

{
  "used": 3,
  "remaining": 2,
  "limit": 5,
  "reset_in_hours": 23,
  "reset_in_minutes": 45
}

GET /api/v1/usage-stats

Get your usage statistics

Headers

Authorization: Bearer your_jwt_token

Response

{
  "today": 3,
  "this_month": 45,
  "total": 120,
  "limit": 10
}

Rate Limits

  • Demo: 5 requests per IP per day
  • Free tier: 10 requests per user per day
  • Pro tier: Unlimited

Code Examples

JavaScript

const response = await fetch('/api/v1/demo-analyze', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    player1_name: 'Player One',
    player1_birthdate: '1990-01-01',
    player2_name: 'Player Two',
    player2_birthdate: '1995-06-15',
    match_date: '2026-02-28',
    sport: 'tennis'
  })
});

const data = await response.json();
console.log(data.winner_prediction);

Python

import requests

response = requests.post('/api/v1/demo-analyze', json={
    'player1_name': 'Player One',
    'player1_birthdate': '1990-01-01',
    'player2_name': 'Player Two',
    'player2_birthdate': '1995-06-15',
    'match_date': '2026-02-28',
    'sport': 'tennis'
})

data = response.json()
print(data['winner_prediction'])
